This code was created in Java, JDK 1.8, using the Eclipse IDE.

It consists of 2 Java files, "DecisionMaker.java" and "TreeNode.java", with DecisionMaker.java containing the main method.

To compile the code, use the command: javac DecisionMaker.java TreeNode.java
To run the code, use the command: java DecisionMaker L K training_set validation_set testing_set yes/no

where:
L = parameter value of L used in post pruning
K = parameter value of K used in post pruning
training_set = path of the training set to be used
validation_set = path of the validation set
testing_set = path of the testing set
yes/no = the words "yes" or "no" to print the decision tree or not
